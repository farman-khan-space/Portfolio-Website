Portfolio Workflow Guide

Your complete guide to setting up, developing, and deploying the website.

  • Install Node.js: nodejs.org
  • Install Git: git-scm.com
npm install -g netlify-cli
git clone <your-repo-url>
cd Portfolio-Website
npm install

For a fast, lightweight server focused only on Eleventy:

npx @11ty/eleventy --serve

To fully simulate the Netlify environment (including forms, redirects):

netlify dev
  • Eleventy dev server runs at: http://localhost:8080
  • Netlify dev server runs at: http://localhost:8888
  • Edit your files inside the src/ directory.
  • Check changes locally using npx @11ty/eleventy --serve.
  • Test any Netlify-specific features with netlify dev.
  • Commit and push your changes to GitHub:
git add .
git commit -m "Your descriptive message"
git push

To commit every changed file individually, you can use this script. Create a file named git-commit-all.sh in your project's root folder.

#!/bin/bash
git status --porcelain | while read -r line; do
  status=$(echo "$line" | awk '{print $1}')
  file=$(echo "$line" | awk '{print $2}')
  commit_message=""
  case "$status" in
    "M") commit_message="style: Update ${file}";;
    "A") commit_message="feat: Add ${file}";;
    "??") commit_message="feat: Add untracked file ${file}";;
    "D") commit_message="fix: Remove ${file}";;
    *) continue;;
  esac
  echo "Staging -> ${file}"
  git add "$file"
  echo "Committing -> ${commit_message}"
  git commit -m "$commit_message"
  echo "-----------------------------------"
done
echo "All files have been committed individually."

How to use on Windows:

  • Right-click in your project folder and select "Git Bash Here".
  • Make the script executable (only needs to be done once):
chmod +x git-commit-all.sh
  • Run the script to commit all files:
./git-commit-all.sh

To manually build and deploy your site from the command line:

npx @11ty/eleventy
netlify deploy --prod --dir=_site --build false

For a faster workflow, add this script to your package.json:

"scripts": {
  "publish": "npx @11ty/eleventy && netlify deploy --prod --dir=_site --build false"
}

Now you can just run one command to deploy:

npm run publish
